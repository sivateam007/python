கோப்பை நீக்குதல்

கோப்பை நீக்க, OS தொகுதியை இறக்குமதி செய்து, அதன் os.remove() செயல்பாட்டை இயக்க வேண்டும்:

எடுத்துக்காட்டு

"demofile.txt" கோப்பை நீக்கவும்:

import os
os.remove("demofile.txt")

கோப்பு இருக்கிறதா என்று சரிபார்க்கவும்:

பிழையைத் தவிர்க்க, கோப்பை நீக்க முயற்சிப்பதற்கு முன் கோப்பு இருக்கிறதா என்று சரிபார்க்க விரும்பலாம்:

எடுத்துக்காட்டு

கோப்பு இருக்கிறதா என்று சரிபார்க்கவும், பின்னர் அதை நீக்கவும்:

import os
if os.path.exists("demofile.txt"):
  os.remove("demofile.txt")
else:
  print("The file does not exist")

கோப்புறையை நீக்குதல்

முழு கோப்புறையையும் நீக்க, os.rmdir() முறையைப் பயன்படுத்தவும்:

எடுத்துக்காட்டு

"myfolder" கோப்புறையை நீக்கவும்:

import os
os.rmdir("myfolder")

⚠️ குறிப்பு:

வெற்றுக் கோப்புறைகளை மட்டுமே நீக்க முடியும்.

பயிற்சி

os.delete()
✗ தவறு! os.delete() என்பது Python இல் கோப்புகளை நீக்க பயன்படுத்தப்படாது
os.drop()
✗ தவறு! os.drop() என்பது Python இல் கோப்புகளை நீக்க பயன்படுத்தப்படாது
os.remove()
✓ சரி! os.remove() செயல்பாடு Python இல் கோப்புகளை நீக்க பயன்படுகிறது
