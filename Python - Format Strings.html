Python - Format - Strings

String Format

As we learned in the Python Variables chapter, we cannot combine strings and numbers like this:

பைத்தான் மாறிகள் அத்தியாயத்தில் நாம் கற்றுக்கொண்டது போல, சரங்களையும் எண்களையும் இப்படி இணைக்க முடியாது:

ExampleGet your own Python Server

age = 36
#This will produce an error:
txt = "My name is John, I am " + age
print(txt)

But we can combine strings and numbers by using f-strings or the format() method!

ஆனால் f-strings அல்லது format() முறையைப் பயன்படுத்தி சரங்களையும் எண்களையும் இணைக்கலாம்!

F-Strings

F-String was introduced in Python 3.6, and is now the preferred way of formatting strings.

F-String பைத்தான் 3.6 இல் அறிமுகப்படுத்தப்பட்டது, இப்போது சரங்களை வடிவமைக்க விரும்பிய வழியாக உள்ளது.

To specify a string as an f-string, simply put an f in front of the string literal, and add curly brackets {} as placeholders for variables and other operations.

ஒரு சரத்தை f-string ஆகக் குறிப்பிட, சரம் மாறிலியின் முன்னால் f வைத்து, மாறிகள் மற்றும் பிற செயல்பாடுகளுக்கான இடக்குறிப்புகளாக சுருள் அடைப்புக்குறிகள் {} சேர்க்கவும்.

Example

Create an f-string:

age = 36
txt = f"My name is John, I am {age}"
print(txt)

Placeholders and Modifiers

A placeholder can contain variables, operations, functions, and modifiers to format the value.

ஒரு இடக்குறிப்பு மதிப்பை வடிவமைக்க மாறிகள், செயல்பாடுகள், செயல்பாடுகள் மற்றும் மாற்றிகள் ஆகியவற்றைக் கொண்டிருக்கலாம்.

Example

Add a placeholder for the price variable:

price = 59
txt = f"The price is {price} dollars"
print(txt)

A placeholder can include a modifier to format the value.

மதிப்பை வடிவமைக்க ஒரு இடக்குறிப்பில் ஒரு மாற்றியைச் சேர்க்கலாம்.

A modifier is included by adding a colon : followed by a legal formatting type, like .2f which means fixed point number with 2 decimals:

ஒரு மாற்றி ஒரு பெருங்குடல் : சேர்ப்பதன் மூலம் சேர்க்கப்பட்டு, அதைத் தொடர்ந்து சட்டப்பூர்வ வடிவமைப்பு வகை, .2f என்பது 2 தசமங்களுடன் நிலையான புள்ளி எண்ணைக் குறிக்கிறது:

Example

Display the price with 2 decimals:

price = 59
txt = f"The price is {price:.2f} dollars"
print(txt)

A placeholder can contain Python code, like math operations:

ஒரு இடக்குறிப்பில் கணித செயல்பாடுகள் போன்ற பைத்தான் குறியீட்டைக் கொண்டிருக்கலாம்:

Example

Perform a math operation in the placeholder, and return the result:

txt = f"The price is {20 * 59} dollars"
print(txt)

Exercise

If x = 9, what is a correct syntax to print 'The price is 9.00 dollars'?

x = 9 எனில், 'The price is 9.00 dollars' என அச்சிடுவதற்கான சரியான தொடரியல் எது?

❓ x = 9 எனில், 'The price is 9.00 dollars' என அச்சிடுவதற்கான சரியான தொடரியல் எது?

print(f'The price is {x:.2f} dollars')
✓ சரி! :.2f மாற்றி மதிப்பை 2 தசம இடங்களுடன் காட்டுகிறது
print(f'The price is {x:2} dollars')
✗ தவறு! :2 மாற்றி தசம இடங்களைக் காட்டாது, அகலத்தை மட்டும் குறிப்பிடுகிறது
print(f'The price is {x:format(2)} dollars')
✗ தவறு! format() செயல்பாடு f-string இடக்குறிப்பில் நேரடியாகப் பயன்படுத்த முடியாது
