String Methods

Python க்கு உள்ளமைக்கப்பட்ட முறைகளின் தொகுப்பு உள்ளது, அவற்றை நீங்கள் சரங்களில் பயன்படுத்தலாம்.

குறிப்பு: அனைத்து சர முறைகளும் புதிய மதிப்புகளைத் தருகின்றன. அவை அசல் சரத்தை மாற்றாது.

முறை விளக்கம்
capitalize() முதல் எழுத்தைப் பெரிய எழுத்தாக மாற்றுகிறது
casefold() சரத்தைச் சிறிய எழுத்துகளாக மாற்றுகிறது
center() மையமாக்கப்பட்ட சரத்தைத் தருகிறது
count() குறிப்பிட்ட மதிப்பு சரத்தில் எத்தனை முறை நிகழ்கிறது என்பதைத் தருகிறது
encode() சரத்தின் குறியாக்கப்பட்ட பதிப்பைத் தருகிறது
endswith() சரம் குறிப்பிட்ட மதிப்புடன் முடிந்தால் உண்மையைத் தருகிறது
expandtabs() சரத்தின் தாவல் அளவை அமைக்கிறது
find() சரத்தில் குறிப்பிட்ட மதிப்பைத் தேடுகிறது மற்றும் அது கண்டறியப்பட்ட இடத்தின் நிலையைத் தருகிறது
format() சரத்தில் குறிப்பிட்ட மதிப்புகளை வடிவமைக்கிறது
format_map() சரத்தில் குறிப்பிட்ட மதிப்புகளை வடிவமைக்கிறது
index() சரத்தில் குறிப்பிட்ட மதிப்பைத் தேடுகிறது மற்றும் அது கண்டறியப்பட்ட இடத்தின் நிலையைத் தருகிறது
isalnum() சரத்தில் உள்ள அனைத்து எழுத்துகளும் எண்ணெழுத்துகளாக இருந்தால் True ஐத் தருகிறது
isalpha() சரத்தில் உள்ள அனைத்து எழுத்துகளும் எழுத்துக்களாக இருந்தால் True ஐத் தருகிறது
isascii() சரத்தில் உள்ள அனைத்து எழுத்துகளும் ascii எழுத்துகளாக இருந்தால் True ஐத் தருகிறது
isdecimal() சரத்தில் உள்ள அனைத்து எழுத்துகளும் தசம எண்களாக இருந்தால் True ஐத் தருகிறது
isdigit() சரத்தில் உள்ள அனைத்து எழுத்துகளும் எண்களாக இருந்தால் True ஐத் தருகிறது
isidentifier() சரம் ஒரு அடையாளங்காட்டியாக இருந்தால் True ஐத் தருகிறது
islower() சரத்தில் உள்ள அனைத்து எழுத்துகளும் சிறிய எழுத்துகளாக இருந்தால் True ஐத் தருகிறது
isnumeric() சரத்தில் உள்ள அனைத்து எழுத்துகளும் எண்ணியல் எழுத்துகளாக இருந்தால் True ஐத் தருகிறது
isprintable() சரத்தில் உள்ள அனைத்து எழுத்துகளும் அச்சிடக்கூடியவையாக இருந்தால் True ஐத் தருகிறது
isspace() சரத்தில் உள்ள அனைத்து எழுத்துகளும் வெள்ளை இட எழுத்துகளாக இருந்தால் True ஐத் தருகிறது
istitle() சரம் தலைப்பின் விதிகளைப் பின்பற்றினால் True ஐத் தருகிறது
isupper() சரத்தில் உள்ள அனைத்து எழுத்துகளும் பெரிய எழுத்துகளாக இருந்தால் True ஐத் தருகிறது
join() ஒரு மறு செய்கை பொருளின் உறுப்புகளை சரத்தின் இறுதியில் இணைக்கிறது
ljust() இடது சீரமைக்கப்பட்ட சரப் பதிப்பைத் தருகிறது
lower() சரத்தைச் சிறிய எழுத்துகளாக மாற்றுகிறது
lstrip() இடது வெட்டப்பட்ட சரப் பதிப்பைத் தருகிறது
maketrans() மொழிபெயர்ப்புகளில் பயன்படுத்த மொழிபெயர்ப்பு அட்டவணையைத் தருகிறது
partition() சரம் மூன்று பகுதிகளாகப் பிரிக்கப்பட்ட tuple ஐத் தருகிறது
replace() குறிப்பிட்ட மதிப்பு மற்றொரு குறிப்பிட்ட மதிப்புடன் மாற்றப்பட்ட சரத்தைத் தருகிறது
rfind() சரத்தில் குறிப்பிட்ட மதிப்பைத் தேடுகிறது மற்றும் அது கடைசியாக கண்டறியப்பட்ட இடத்தின் நிலையைத் தருகிறது
rindex() சரத்தில் குறிப்பிட்ட மதிப்பைத் தேடுகிறது மற்றும் அது கடைசியாக கண்டறியப்பட்ட இடத்தின் நிலையைத் தருகிறது
rjust() வலது சீரமைக்கப்பட்ட சரப் பதிப்பைத் தருகிறது
rpartition() சரம் மூன்று பகுதிகளாகப் பிரிக்கப்பட்ட tuple ஐத் தருகிறது
rsplit() குறிப்பிட்ட பிரிப்பானில் சரத்தைப் பிரிக்கிறது, மேலும் பட்டியலைத் தருகிறது
rstrip() வலது வெட்டப்பட்ட சரப் பதிப்பைத் தருகிறது
split() குறிப்பிட்ட பிரிப்பானில் சரத்தைப் பிரிக்கிறது, மேலும் பட்டியலைத் தருகிறது
splitlines() வரி முறிவுகளில் சரத்தைப் பிரிக்கிறது, மேலும் பட்டியலைத் தருகிறது
startswith() சரம் குறிப்பிட்ட மதிப்புடன் தொடங்கினால் உண்மையைத் தருகிறது
strip() வெட்டப்பட்ட சரப் பதிப்பைத் தருகிறது
swapcase() வழக்குகளை மாற்றுகிறது, சிறிய எழுத்து பெரிய எழுத்தாகவும், பெரிய எழுத்து சிறிய எழுத்தாகவும் மாறுகிறது
title() ஒவ்வொரு சொல்லின் முதல் எழுத்தையும் பெரிய எழுத்தாக மாற்றுகிறது
translate() மொழிபெயர்க்கப்பட்ட சரத்தைத் தருகிறது
upper() சரத்தைப் பெரிய எழுத்துகளாக மாற்றுகிறது
zfill() சரத்தின் தொடக்கத்தில் குறிப்பிட்ட எண்ணிக்கையிலான 0 மதிப்புகளை நிரப்புகிறது
