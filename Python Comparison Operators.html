Comparison Operators

ஒப்பீட்டு ஆபரேட்டர்கள் இரண்டு மதிப்புகளை ஒப்பிட பயன்படுகின்றன:

Operator Name Example
== Equal x == y
!= Not equal x != y
> Greater than x > y
< Less than x < y
>= Greater than or equal to x >= y
<= Less than or equal to x <= y

Examples

ஒப்பீட்டு ஆபரேட்டர்கள் ஒப்பீட்டின் அடிப்படையில் True அல்லது False ஐத் தருகின்றன:

Example

x = 5
y = 3

print(x == y)
print(x != y)
print(x > y)
print(x < y)
print(x >= y)
print(x <= y)

Chaining Comparison Operators

பைத்தான் ஒப்பீட்டு ஆபரேட்டர்களை சங்கிலியாக அனுமதிக்கிறது:

Example

x = 5

print(1 < x < 10)

print(1 < x and x < 10)
