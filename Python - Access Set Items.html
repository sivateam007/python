Python - Access Set Items

Set உருப்படிகளை அணுகுதல்

உருப்படிகளை அணுகுதல்

அட்டவணை அல்லது விசையைக் குறிப்பிடுவதன் மூலம் set இல் உள்ள உருப்படிகளை அணுக முடியாது.

ஆனால் for loop ஐப் பயன்படுத்தி set உருப்படிகள் வழியாக சுழலலாம், அல்லது in முக்கியச்சொல்லைப் பயன்படுத்தி குறிப்பிட்ட மதிப்பு set இல் உள்ளதா என்று கேட்கலாம்.

எடுத்துக்காட்டு

Set வழியாகச் சுழன்று மதிப்புகளை அச்சிடவும்:

thisset = {"apple", "banana", "cherry"}

for x in thisset:
  print(x)

எடுத்துக்காட்டு

"banana" set இல் உள்ளதா என்று சரிபார்க்கவும்:

thisset = {"apple", "banana", "cherry"}

print("banana" in thisset)

எடுத்துக்காட்டு

"banana" set இல் இல்லையா என்று சரிபார்க்கவும்:

thisset = {"apple", "banana", "cherry"}

print("banana" not in thisset)

உருப்படிகளை மாற்றுதல்

Set உருவாக்கப்பட்ட பிறகு, அதன் உருப்படிகளை மாற்ற முடியாது, ஆனால் புதிய உருப்படிகளைச் சேர்க்கலாம்.

⚠️ குறிப்பு:

Sets மாற்ற முடியாதவை, எனவே set உருவாக்கப்பட்ட பிறகு இருக்கும் உருப்படிகளை மாற்ற முடியாது. ஆனால் உருப்படிகளை நீக்கலாம் மற்றும் புதிய உருப்படிகளைச் சேர்க்கலாம்.

பயிற்சி

❓ உண்மை அல்லது பொய். அட்டவணையைக் குறிப்பிடுவதன் மூலம் set உருப்படிகளை அணுக முடியும்.

உண்மை
✗ தவறு! Sets அட்டவணைப்படுத்தப்படாதவை, எனவே அட்டவணை எண்ணைப் பயன்படுத்தி set உருப்படிகளை நேரடியாக அணுக முடியாது
பொய்
