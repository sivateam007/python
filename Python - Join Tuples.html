Python - Join Tuples

Tuples ஐ இணைத்தல்

இரண்டு Tuples ஐ இணைத்தல்

இரண்டு அல்லது அதற்கு மேற்பட்ட tuples ஐ இணைக்க + ஆபரேட்டரைப் பயன்படுத்தலாம்:

எடுத்துக்காட்டு

இரண்டு tuples ஐ இணைத்தல்:

tuple1 = ("a", "b" , "c")
tuple2 = (1, 2, 3)

tuple3 = tuple1 + tuple2
print(tuple3)

Tuples ஐ பெருக்குதல்

ஒரு tuple இன் உள்ளடக்கத்தை கொடுக்கப்பட்ட எண்ணிக்கையில் பெருக்க விரும்பினால், * ஆபரேட்டரைப் பயன்படுத்தலாம்:

எடுத்துக்காட்டு

fruits tuple ஐ 2 ஆல் பெருக்கவும்:

fruits = ("apple", "banana", "cherry")
mytuple = fruits * 2

print(mytuple)

பயிற்சி

❓ tuple1 மற்றும் tuple2 ஐ tuple3 ஆக இணைப்பதற்கான சரியான தொடரியல் எது?

tuple3 = join(tuple1, tuple2)
✗ தவறு! Python இல் join() என்பது தனித்துவமான tuple முறை அல்ல
tuple3 = tuple1 + tuple2
✓ சரி! + ஆபரேட்டர் Python இல் tuples ஐ இணைக்கப் பயன்படுகிறது
tuple3 = [tuple1, tuple2]
✗ தவறு! இது tuple உருவாக்காது, இரண்டு tuples கொண்ட பட்டியலை உருவாக்கும்
