பைத்தான் - அகராதி உருப்படிகளைச் சேர்த்தல்

உருப்படிகளைச் சேர்த்தல்

அகராதியில் ஒரு உருப்படியைச் சேர்ப்பது ஒரு புதிய குறியீட்டு விசையைப் பயன்படுத்தி அதற்கு ஒரு மதிப்பைக் குறிப்பிடுவதன் மூலம் செய்யப்படுகிறது:

எடுத்துக்காட்டு

thisdict = {
  "brand": "Ford",
  "model": "Mustang",
  "year": 1964
}
thisdict["color"] = "red"
print(thisdict)

அகராதியைப் புதுப்பித்தல்

update() முறை கொடுக்கப்பட்ட வாதத்திலிருந்து உருப்படிகளுடன் அகராதியைப் புதுப்பிக்கும். உருப்படி இல்லையென்றால், உருப்படி சேர்க்கப்படும்.

வாதம் ஒரு அகராதியாக இருக்க வேண்டும், அல்லது key:value ஜோடிகளுடன் கூடிய மீள் செயல்படுத்தக்கூடிய பொருளாக இருக்க வேண்டும்.

எடுத்துக்காட்டு

update() முறையைப் பயன்படுத்தி அகராதியில் color உருப்படியைச் சேர்:

thisdict = {
  "brand": "Ford",
  "model": "Mustang",
  "year": 1964
}
thisdict.update({"color": "red"})

பயிற்சி

❓ இந்த அகராதி முறைகளில் எதை அகராதியில் உருப்படிகளைச் சேர்க்கப் பயன்படுத்தலாம்?

add()
✗ தவறு! add() என்பது அகராதி முறை அல்ல
insert()
✗ தவறு! insert() என்பது பட்டியல் முறை, அகராதி முறை அல்ல
update()
✓ சரி! update() முறை அகராதியில் உருப்படிகளைச் சேர்க்க அல்லது புதுப்பிக்கப் பயன்படுகிறது
