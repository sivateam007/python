append() vs extend() Confusion

# Wrong: Trying to append a list
my_list = [1, 2, 3]
my_list.append([4, 5])
# Result: [1, 2, 3, [4, 5]] (nested list)

# Correct: Use extend()
my_list = [1, 2, 3]
my_list.extend([4, 5])
# Result: [1, 2, 3, 4, 5]
