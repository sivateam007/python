NumPy Tutorial

NumPy ஐக் கற்றுக்கொள்ளுங்கள்

NumPy டுடோரியல்

NumPy என்பது பைதான் நூலகமாகும்.

NumPy வரிசைகளுடன் பணிபுரியப் பயன்படுகிறது.

NumPy என்பது "Numerical Python" என்பதன் சுருக்கமாகும்.

வாசிப்பதன் மூலம் கற்றல்

NumPy பற்றி மேலும் அறிய உங்களுக்காக 43 டுடோரியல் பக்கங்களை உருவாக்கியுள்ளோம்.

ஒரு அடிப்படை அறிமுகத்துடன் தொடங்கி, சீரற்ற தரவுத் தொகுப்புகளை உருவாக்குதல் மற்றும் வரைபடமாக்குதல், மற்றும் NumPy செயல்பாடுகளுடன் பணிபுரிதல் ஆகியவற்றுடன் முடிகிறது:

எடுத்துக்காட்டுகள் மூலம் கற்றல்

எங்கள் "Try it Yourself" தொகுப்பியில், NumPy தொகுதியைப் பயன்படுத்தலாம், மேலும் முடிவைப் பார்க்க குறியீட்டை மாற்றலாம்.

எடுத்துக்காட்டு

NumPy வரிசையை உருவாக்கவும்:

import numpy as np

arr = np.array([1, 2, 3, 4, 5])

print(arr)

print(type(arr))

இது எவ்வாறு செயல்படுகிறது என்பதைப் பார்க்க "Try it Yourself" பொத்தானைக் கிளிக் செய்யவும்.
