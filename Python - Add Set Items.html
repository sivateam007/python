Python - Add Set Items

Set உருப்படிகளைச் சேர்த்தல்

உருப்படிகளைச் சேர்த்தல்

Set உருவாக்கப்பட்ட பிறகு, அதன் உருப்படிகளை மாற்ற முடியாது, ஆனால் புதிய உருப்படிகளைச் சேர்க்கலாம்.

Set க்கு ஒரு உருப்படியைச் சேர்க்க add() முறையைப் பயன்படுத்தவும்.

எடுத்துக்காட்டு

add() முறையைப் பயன்படுத்தி set க்கு உருப்படியைச் சேர்க்கவும்:

thisset = {"apple", "banana", "cherry"}

thisset.add("orange")

print(thisset)

Sets ஐச் சேர்த்தல்

மற்றொரு set இலிருந்து உருப்படிகளை தற்போதைய set க்குச் சேர்க்க, update() முறையைப் பயன்படுத்தவும்.

எடுத்துக்காட்டு

tropical இலிருந்து உறுப்புகளை thisset க்குச் சேர்க்கவும்:

thisset = {"apple", "banana", "cherry"}
tropical = {"pineapple", "mango", "papaya"}

thisset.update(tropical)

print(thisset)

எந்த மீண்டும் மீண்டும் செய்யக்கூடிய பொருளையும் சேர்த்தல்

update() முறையில் உள்ள பொருள் set ஆக இருக்க வேண்டியதில்லை, அது எந்த மீண்டும் மீண்டும் செய்யக்கூடிய பொருளாகவும் இருக்கலாம் (tuples, lists, dictionaries போன்றவை).

எடுத்துக்காட்டு

பட்டியலின் உறுப்புகளை set க்குச் சேர்க்கவும்:

thisset = {"apple", "banana", "cherry"}
mylist = ["kiwi", "orange"]

thisset.update(mylist)

print(thisset)

பயிற்சி

❓ set க்கு உருப்படிகளைச் சேர்ப்பதற்கான சரியான தொடரியல் எது?

add()
✓ சரி! add() முறை set க்கு ஒரு உருப்படியைச் சேர்க்கப் பயன்படுகிறது
insert()
✗ தவறு! insert() முறை பட்டியல்களுக்குப் பயன்படுகிறது, sets க்கு அல்ல
push()
✗ தவறு! push() முறை sets க்குப் பயன்படுத்தப்படாது, மற்ற நிரலாக்க மொழிகளில் பயன்படுத்தப்படலாம்
