Escape Character

சரத்தில் சட்டவிரோதமான எழுத்துக்களைச் செருக, தப்பு எழுத்தைப் பயன்படுத்தவும்.

ஒரு தப்பு எழுத்து என்பது நீங்கள் செருக விரும்பும் எழுத்துக்குப் பின்னால் ஒரு பின்சாய்வு \ ஆகும்.

சட்டவிரோத எழுத்துக்கான உதாரணம் இரட்டை மேற்கோள்களால் சூழப்பட்ட ஒரு சரத்திற்குள் இரட்டை மேற்கோளாகும்:

Example

txt = "We are the so-called "Vikings" from the north."

இந்தச் சிக்கலை சரிசெய்ய, தப்பு எழுத்து \" ஐப் பயன்படுத்தவும்:

Example

txt = "We are the so-called \"Vikings\" from the north."

Escape Characters

பைத்தானில் பயன்படுத்தப்படும் பிற தப்பு எழுத்துக்கள்:

Code Result
\' Single Quote
\\ Backslash
\n New Line
\r Carriage Return
\t Tab
\b Backspace
\f Form Feed
\ooo Octal value
\xhh Hex value

Escape Character Examples

"
Double Quotes in String
txt = "He said, \"Hello!\""

இரட்டை மேற்கோள்களுக்குள் இரட்டை மேற்கோள்களைச் செருக

↵
New Line
txt = "First line\nSecond line"

சரத்தில் புதிய வரியைச் செருக

→
Tab Space
txt = "Name:\tJohn"

சரத்தில் தட்டுப்படி இடத்தைச் செருக

\
Backslash in String
path = "C:\\Users\\John"

விண்டோஸ் பாதைகளில் பின்சாய்வுகளைச் செருக

Exercise

❓ பின்வரும் எந்த தப்பு எழுத்து சரத்தில் புதிய வரியைச் செருக பயன்படுகிறது?

\t
✗ தவறு! \t என்பது தட்டுப்படி இடத்திற்குப் பயன்படுகிறது, புதிய வரிக்கு அல்ல
\n
✓ சரி! \n என்பது பைத்தானில் புதிய வரி எழுத்துக்குப் பயன்படுகிறது
\r
✗ தவறு! \r என்பது கேரியேஜ் ரிட்டர்னுக்குப் பயன்படுகிறது, புதிய வரிக்கு அல்ல
\b
✗ தவறு! \b என்பது பேக்ஸ்பேஸுக்குப் பயன்படுகிறது, புதிய வரிக்கு அல்ல
