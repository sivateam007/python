Python Variables

பைத்தான் மாறிகள்

Variables

மாறிகள் என்பது தரவு மதிப்புகளைச் சேமிக்கும் கொள்கலன்கள்.

Creating Variables

பைத்தானில் மாறியை அறிவிக்க எந்த கட்டளையும் இல்லை.

நீங்கள் முதல் முறையாக ஒரு மதிப்பை ஒதுக்கும் தருணத்தில் ஒரு மாறி உருவாக்கப்படுகிறது.

Example

x = 5
y = "John"
print(x)
print(y)

மாறிகள் எந்த குறிப்பிட்ட வகையுடனும் அறிவிக்கப்பட வேண்டியதில்லை, மேலும் அவை அமைக்கப்பட்ட பிறகும் வகையை மாற்ற முடியும்.

Example

x = 4       # x is of type int
x = "Sally" # x is now of type str
print(x)

Casting

ஒரு மாறியின் தரவு வகையைக் குறிப்பிட விரும்பினால், இது காஸ்டிங்க் மூலம் செய்யப்படலாம்.

Example

x = str(3)    # x will be '3'
y = int(3)    # y will be 3
z = float(3)  # z will be 3.0

Get the Type

type() செயல்பாட்டைப் பயன்படுத்தி ஒரு மாறியின் தரவு வகையைப் பெறலாம்.

Example

x = 5
y = "John"
print(type(x))
print(type(y))

இந்த டுடோரியலில் பின்னர் தரவு வகைகள் மற்றும் காஸ்டிங் பற்றி மேலும் அறிவீர்கள்.

Single or Double Quotes?

சரம் மாறிகள் ஒற்றை அல்லது இரட்டை மேற்கோள்களைப் பயன்படுத்தி அறிவிக்கப்படலாம்:

Example

x = "John"
# is the same as
x = 'John'

Case-Sensitive

மாறி பெயர்கள் வழக்கு-உணர்திறன் கொண்டவை.

Example

a = 4
A = "Sally"
#A will not overwrite a

Exercise

❓ பைத்தான் மாறியை அறிவிக்க சரியான வழி எது?

var x = 5
✗ தவறு! பைத்தானில் var விசேஷ சொல் தேவையில்லை
#x = 5
✗ தவறு! # என்பது குறிப்பு (comment) ஆகும், இது குறியீட்டை இயக்காது
$x = 5
✗ தவறு! $ என்பது பைத்தான் மாறி பெயர்களில் அனுமதிக்கப்படாது
x = 5
✓ சரி! இது பைத்தானில் மாறியை உருவாக்குவதற்கான சரியான தொடரியல்
