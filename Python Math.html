Python Math

Python கட்டமைக்கப்பட்ட கணித செயல்பாடுகளின் தொகுப்பைக் கொண்டுள்ளது, இதில் ஒரு விரிவான math மாட்யூலும் அடங்கும், இது எண்களில் கணிதப் பணிகளைச் செய்ய உங்களை அனுமதிக்கிறது.

கட்டமைக்கப்பட்ட கணித செயல்பாடுகள்

min() மற்றும் max() செயல்பாடுகளை ஒரு iterable இல் மிகக் குறைந்த அல்லது மிக உயர்ந்த மதிப்பைக் கண்டறிய பயன்படுத்தலாம்:

Example

x = min(5, 10, 25)
y = max(5, 10, 25)

print(x)
print(y)

abs() செயல்பாடு குறிப்பிடப்பட்ட எண்ணின் முழுமையான (நேர்மறை) மதிப்பை வழங்குகிறது:

Example

x = abs(-7.25)

print(x)

pow(x, y) செயல்பாடு x இன் மதிப்பை y (x^y) இன் சக்திக்கு வழங்குகிறது.

Example

4 இன் மதிப்பை 3 இன் சக்திக்கு வழங்கவும் (4 * 4 * 4 போன்றது):

x = pow(4, 3)

print(x)

Math மாட்யூல்

Python கட்டமைக்கப்பட்ட math என்ற மாட்யூலையும் கொண்டுள்ளது, இது கணித செயல்பாடுகளின் பட்டியலை விரிவுபடுத்துகிறது.

அதைப் பயன்படுத்த, நீங்கள் math மாட்யூலை இறக்குமதி செய்ய வேண்டும்:

import math

நீங்கள் math மாட்யூலை இறக்குமதி செய்தவுடன், மாட்யூலின் முறைகள் மற்றும் மாறிலிகளைப் பயன்படுத்தத் தொடங்கலாம்.

math.sqrt() முறை எடுத்துக்காட்டாக, ஒரு எண்ணின் வர்க்க மூலத்தை வழங்குகிறது:

Example

import math

x = math.sqrt(64)

print(x)

math.ceil() முறை ஒரு எண்ணை அதன் அருகிலுள்ள முழு எண்ணுக்கு மேலே சுற்றுகிறது, மேலும் math.floor() முறை ஒரு எண்ணை அதன் அருகிலுள்ள முழு எண்ணுக்கு கீழே சுற்றுகிறது மற்றும் முடிவை வழங்குகிறது:

Example

import math

x = math.ceil(1.4)
y = math.floor(1.4)

print(x) # returns 2
print(y) # returns 1

math.pi மாறிலி, PI இன் மதிப்பை வழங்குகிறது (3.14...):

Example

import math

x = math.pi

print(x)

பயிற்சி

❓ பின்வரும் குறியீட்டைக் கவனியுங்கள்:
print(max(5, 10, 25))
அச்சிடப்பட்ட முடிவு என்னவாக இருக்கும்?

5
✗ தவறு! 5 என்பது மிகக் குறைந்த மதிப்பு, max() செயல்பாடு மிக உயர்ந்த மதிப்பை வழங்குகிறது.
10
✗ தவறு! 10 என்பது நடுத்தர மதிப்பு, max() செயல்பாடு மிக உயர்ந்த மதிப்பை வழங்குகிறது.
25
✓ சரி! max() செயல்பாடு மிக உயர்ந்த மதிப்பை வழங்குகிறது, இது 25 ஆகும்.
