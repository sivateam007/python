Django Tutorial

Django ஐக் கற்றுக்கொள்ளுங்கள்

Django டுடோரியல்

Django என்பது பின்புற சேவையக பக்க வலைக் கட்டமைப்பாகும்.

Django இலவசம், திறந்த மூல மென்பொருள் மற்றும் Python இல் எழுதப்பட்டது.

Django Python ஐப் பயன்படுத்தி வலைப்பக்கங்களை உருவாக்குவதை எளிதாக்குகிறது.

செய்வதன் மூலம் கற்றல்

இந்த டுடோரியலில், Django திட்டத்தை எவ்வாறு நிறுவுவது மற்றும் உருவாக்குவது என்பது பற்றிய படிப்படியான வழிகாட்டியைப் பெறுவீர்கள். தரவுகளைச் சேர்க்க, வாசிக்க, புதுப்பிக்க அல்லது நீக்கக்கூடிய ஒரு திட்டத்தை எவ்வாறு உருவாக்குவது என்பதை நீங்கள் கற்றுக் கொள்வீர்கள்.

HTML டெம்ப்ளேட்டுகளை எவ்வாறு உருவாக்குவது மற்றும் HTML ஆவணத்திற்குள் தரவைச் செருக Django டெம்ப்ளேட் குறிச்சொற்களை எவ்வாறு பயன்படுத்துவது என்பதை நீங்கள் கற்றுக் கொள்வீர்கள்.

தரவுத்தளத்திலிருந்து தரவைப் பிரித்தெடுக்க, வடிகட்டி மற்றும் வரிசைப்படுத்த QuerySets உடன் எவ்வாறு பணிபுரிவது என்பதை நீங்கள் கற்றுக் கொள்வீர்கள்.

PostgreSQL தரவுத்தளத்தை எவ்வாறு அமைப்பது மற்றும் உங்கள் Django திட்டத்தை உலகிற்கு எவ்வாறு வெளியிடுவது என்பதையும் நீங்கள் கற்றுக் கொள்வீர்கள்.

எடுத்துக்காட்டுகள் மூலம் கற்றல்

டுடோரியலில், பல்வேறு கருத்துகளை சிறப்பாக விளக்குவதற்கு எடுத்துக்காட்டுகளைப் பயன்படுத்துவோம்.

எடுத்துக்காட்டு

<ul>
  {% for x in mymembers %}
    <li>{{ x.firstname }}</li>
  {% endfor %}
</ul>

பயிற்சிகள் மூலம் கற்றல்

இந்த டுடோரியலில் உள்ள பல அத்தியாயங்கள் உங்கள் அறிவு நிலையைச் சரிபார்க்கக்கூடிய பயிற்சியுடன் முடிகின்றன.

❓ Django திட்டத்தை இயக்குவதற்கான சரியான தொடரியல் எது?

python manage.py startserver
✗ தவறு! இது சரியான கட்டளை அல்ல
python manage.py runserver
✓ சரி! இது Django சேவையகத்தை இயக்குவதற்கான சரியான கட்டளையாகும்
python manage.py start
✗ தவறு! இது சரியான கட்டளை அல்ல
