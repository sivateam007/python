பைத்தான் வெளியீட்டு எண்கள்

Print Numbers

எண்களைக் காட்டவும் print() செயல்பாட்டைப் பயன்படுத்தலாம்:

இருப்பினும், உரையைப் போலல்லாமல், எண்களை இரட்டை மேற்கோள்களுக்குள் வைக்க மாட்டோம்:

Example

print(3)
print(358)
print(50000)

print() செயல்பாட்டிற்குள் கணிதத்தையும் செய்யலாம்:

Example

print(3 + 3)
print(2 * 5)

Mix Text and Numbers

ஒரு வெளியீட்டில் உரை மற்றும் எண்களை காற்புள்ளியால் பிரித்து இணைக்கலாம்:

Example

print("I am", 35, "years old.")

பைத்தான் மாறிகள் மற்றும் பைத்தான் சரங்கள் அத்தியாயங்களில் உரை மற்றும் மாறிகளை இணைப்பது பற்றி மேலும் அறிவீர்கள்.

Exercise

❓ பின்வருவனவற்றில் எது சரியான print() கூற்று?

print "Hello World"
✗ தவறு! print செயல்பாட்டிற்கு அடைப்புக்குறிகள் தேவை
print("Hello", 5)
✓ சரி! உரை மற்றும் எண்களை காற்புள்ளியால் பிரித்து அச்சிடலாம்
print("5 + 3")
✗ தவறு! இது கணிதத்தை மதிப்பீடு செய்யாது, சரமாக அச்சிடும்
print(5 + 3)
✓ சரி! print() செயல்பாட்டிற்குள் கணிதத்தைச் செய்யலாம்
