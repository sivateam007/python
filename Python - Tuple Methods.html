Tuple முறைகள்

Python இல் இரண்டு உள்ளமைக்கப்பட்ட முறைகள் உள்ளன, அவற்றை நீங்கள் tuples இல் பயன்படுத்தலாம்.

முறை விளக்கம்
count() குறிப்பிட்ட மதிப்பு ஒரு tuple இல் எத்தனை முறை நிகழ்கிறது என்பதை வழங்குகிறது
index() குறிப்பிட்ட மதிப்புக்காக tuple ஐத் தேடுகிறது மற்றும் அது காணப்பட்ட இடத்தை வழங்குகிறது

count() முறை

count() முறை ஒரு tuple இல் ஒரு மதிப்பு எத்தனை முறை நிகழ்கிறது என்பதை வழங்குகிறது.

எடுத்துக்காட்டு

மதிப்பு 5 எத்தனை முறை நிகழ்கிறது என்பதை வழங்கவும்:

thistuple = (1, 3, 7, 8, 7, 5, 4, 6, 8, 5)
x = thistuple.count(5)
print(x)

index() முறை

index() முறை tuple இல் குறிப்பிட்ட மதிப்பைத் தேடுகிறது மற்றும் அது காணப்பட்ட இடத்தை வழங்குகிறது.

எடுத்துக்காட்டு

மதிப்பு 8 முதல் முறையாக எந்த இடத்தில் நிகழ்கிறது என்பதை வழங்கவும்:

thistuple = (1, 3, 7, 8, 7, 5, 4, 6, 8, 5)
x = thistuple.index(8)
print(x)

⚠️ குறிப்பு:

மதிப்பு காணப்படவில்லை என்றால், index() முறை பிழையை எழுப்பும்.
